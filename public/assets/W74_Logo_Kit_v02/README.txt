W74 LOGO KIT
============
The W74 mark: a rounded square containing "W" (Space Grotesk Bold) with a
superscript "74" (IBM Plex Mono) — a nod to tungsten, element 74.

Brand colors:
  Ink (box / dark mark):  #15151C
  White:                  #FFFFFF
  Green accent (optional):#2E7D52

-------------------------------------------------------------------
FILES
-------------------------------------------------------------------
/svg  — vector masters (infinitely scalable; use these wherever possible)
  w74_mark_dark.svg ............ PRIMARY. Black box, white W74. Default everywhere.
  w74_mark_dark_green74.svg .... Accent variant: green "74". Use on web headers / hero.
  w74_mark_white.svg ........... Inverted: white box, ink W74. For dark photo backgrounds
                                 (give it breathing room or a subtle border).
  w74_mark_transparent_ink.svg . No box, ink W74. Flexible placement on light surfaces.
  w74_mark_transparent_white.svg No box, white W74. Flexible placement on dark surfaces.

/png — raster exports (when SVG isn't accepted)
  w74_mark_dark_{32..1024}.png ........ general use, transparent background
  w74_mark_dark_120_email.png ......... drop into an email signature
  w74_mark_white_glyphs_{128..1024}.png white W74, no box — for dark backgrounds
  w74_slack_workspace_512_padded.png .. Slack & Google Workspace avatar (14% safe padding
                                        so round crops don't clip the mark)
  w74_appicon_1024_padded.png ......... app-store / large icon with safe padding
  /favicon
    favicon.ico ....................... multi-size .ico (16/32/48) for websites
    favicon_{16,32,48}.png ............ individual favicon PNGs

-------------------------------------------------------------------
WHERE TO USE WHAT
-------------------------------------------------------------------
Website header / letterhead / docs ... svg/w74_mark_dark.svg (or green74 on hero)
Favicon .............................. png/favicon/favicon.ico
Slack / Google Workspace avatar ...... png/w74_slack_workspace_512_padded.png
Email signature ...................... png/w74_mark_dark_120_email.png (or the SVG)
On dark / photographic backgrounds ... svg/w74_mark_white.svg or white_glyphs PNGs
Social / app icon .................... png/w74_appicon_1024_padded.png

-------------------------------------------------------------------
USAGE NOTES
-------------------------------------------------------------------
• Clear space: keep empty space ≈ 1/4 of the mark's width on all sides.
• Minimum size: 24px (square mark). Below that the "74" loses legibility —
  use the transparent W-only feel only if you make a dedicated tiny variant.
• Don't recolor, stretch, rotate, add effects, or change the W/74 spacing.
• Prefer SVG over PNG anywhere the platform allows it (crisp at every size).
• The square is NOT a perfect circle — for round avatar slots use the
  *_padded.png so the corners aren't clipped awkwardly.

Typeface: Space Grotesk (W) + IBM Plex Mono (74). Both are free via Google Fonts
if you ever need to set "W74" in matching type elsewhere.
